git branch and merge

  • 1. git branch -m feature (-M -> sebagai head, -m sebagai branch)
  • 2. git branch -> melihat branchnya
  • 3. git checkout feature -> berpindah ke branch feature
  • 4. commit project
  • 5. git push -u origin feature
  • 6. git checkout main -> berpindah ke branch main
  • 7. git pull -u origin main -> merge project dari branch feature ke branch main
