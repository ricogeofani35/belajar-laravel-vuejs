push and pull project to github

  • 1. git init
  • 2. git add name_file (. -> semua file)
  • 3. git commit -m 'description commit'
  • 4. git branch -M main -> membuat branch main
  • 5. git remote add origin https://github.com/ricogeofani35/belajar-laravel-vuejs.git -> remote github
  • 6. git push -u origin main -> push project (mendorong project ke github)
  • 7. git pull -u origin main -> pull project (menarik project dari github)
